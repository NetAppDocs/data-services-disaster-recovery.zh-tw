= 使用NetApp Disaster Recovery概述
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/use/


[role="lead"]
使用NetApp Disaster Recovery，您可以實現以下目標：

* link:../use/dashboard-view.html["查看災難復原計畫的健康狀況"]
* link:../use/sites-add.html["將資源新增至站台"]
* link:../use/resource-group.html["建立資源群組以組織資料來源"]
* link:../use/drplan-create.html["制定災難復原計劃"]
* link:../use/replicate.html["複製 VMware 應用程式"] 使用 SnapMirror 複製功能，將主站點複製到雲端的災難復原遠端站點
* link:../use/migrate.html["遷移應用程式"] 從您的主站點到另一個站點
* link:../use/failover.html["測試故障轉移"] 在不中斷資源的情況下
* 萬一發生災難，link:../use/failover.html["故障轉移您的主站點"]
* 災難解決後，link:../use/failback.html["故障恢復"]從災難復原站台到主要站台
* link:../use/monitor-jobs.html["監控災難復原操作"] 在 Job Monitoring 頁面上
