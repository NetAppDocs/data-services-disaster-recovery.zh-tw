= 在NetApp Disaster Recovery概述中建立複製計劃
:allow-uri-read: 


[role="lead"]
在本機網站上有要保護的 vCenter 並且您有一個配置為使用Amazon FSx for NetApp ONTAP 的Amazon EVS 站點（可將其用作 DR 目標）後，您可以建立複製計劃 (RP) 來保護本地站點內 vCenter 叢集上託管的任何虛擬機器集。

.若要開始複製計劃創建過程：
. 從任何NetApp Disaster Recovery畫面中，選擇「複製計畫」選項。
+
image:evs-create-rp-1.png["複製計劃選項"]

. 在複製計劃頁面中，選擇*新增*。
+
image:evs-create-rp-2.png["複製計劃畫面"]



這將開啟建立複製計劃精靈。

繼續link:evs-deploy-guide-create-rp-wiz-01.html["建立複製計劃精靈步驟 1"]。
