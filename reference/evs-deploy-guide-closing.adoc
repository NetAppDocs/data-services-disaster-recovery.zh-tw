= 結論
:allow-uri-read: 


[role="lead"]
當配置為使用 Amazon EVS 和Amazon FSx for NetApp ONTAP作為 DR 目標時， NetApp Disaster Recovery可為您的內部部署 VMware 基礎架構提供經濟高效的混合雲 DR 解決方案。這為您提供了針對關鍵任務基礎架構的強大 DR 目標。  NetApp Disaster Recovery利用原生ONTAP資料保護功能，這些功能以數十年的資料保護經驗為後盾，經過深入測試，可確保您的資料得到全面保護。
